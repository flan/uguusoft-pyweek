Angel-G
===============

Entry in PyWeek #9  <http://www.pyweek.org/9/>
Team: Puukusoft-lite
Members: Arashi, Red HamsterX


DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/
  PyGame:     http://www.pygame.org/



RUNNING THE GAME:

On Windows or Mac OS X, locate the "run_game.pyw" file and double-click it.

Othewise open a terminal / console and "cd" to the game directory and run:

  python run_game.py



HOW TO PLAY THE GAME:

Use the arrow keys to move, the space bar to shoot, left CTRL to launch feahter
bombs, and left Shift to dash.

For more information, read the tips on the title screen.
